Chapter6 : 문자열 다루기(String)
문자열(String)

- 시퀀스 자료형으로, 문자형 data를 메모리에 저장
- 영문자 한글자는 8bit = 1byte

import sys
sys.getsizeof("a")
sys.getsizeof("ab")  # 글자가 한글자씩 증가할때 마다, 1byte 씩 증가
                     # 50은 한 문자를 저장하기 위해 사용되는 기본 default size라고 생각하자


- string은 1byte 크기로 한글자씩 메모리 공간이 할당

- 1 Byte의 메모리 공간?
	컴퓨터는 2진수로 데이터 저장
	이진수 한 자리수는 1bit로 저장됨
	즉 1bit 는 0 또는 1
	1 byte = 8 bit = 2^8 = 256 저장 가능

- 컴퓨터는 직접적으로 문자를 인식하지 못함 -> 문자를 2진수로 저장
  이러한 2진수로 저장할 때 표준 규칙을 저장 ex) MS949, UTF-8 등...
 
- 프로그램 언어에서의 데이터 타입
 정수형 int   4byte : -2^31 ~ 2^31-1
 정수형 long 무제한 : 무제한
 실수형 float 8byte 약 10^308 ~ 10^308

- 문자열 인덱싱
- 문자열의 각 문자는 개별 주소(offset)을 가짐
- 이 '주소'를 사용해 할당된 값을 가져오는 것이 인덱싱
- List와 같은 형태로 데이터를 처리

 h  e  l  l o
 0  1  2  3  4
-5 -4 -3 -2 -1

- 단일 값을 호출하는 것은 indexOfBounding 은 error 발생
- 범위를 지정하는 것은 index 가 벗어나도 에러는 발생하지 않는다

- 다양한 문자열 표현
문자열 선언은 큰따옴표("")나 작은 따옴표 ('') 를 활용
