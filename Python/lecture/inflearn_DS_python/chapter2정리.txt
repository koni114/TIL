chapter2 : 자료형 변환
- python 의 특징 : dynamic typing -> 변수 앞에 타입 선언을 하지 않아도
                                                 동적으로 타입을 지정해줌

- integer + integer -> integer 타입으로 return
- 숫자형 -> 문자형 : string(a)
- 숫자형 타입과 문자형 타입은 덧셈이 되지 않음
- 소수점이 존재하는 문자형 데이터를 integer 형으로 변환할 수 없음
- 문자형 + 문자형 -> 더해서 출력
- 컴퓨터의 반올림 오차 -> 3.x 버전에서는 없음.
  c = 38.8
  print(c) : 38.8
  c
  -> 38.7999... 로 변환 됨 -> 3.0 이상 버전에서는 해당 문제 X
- 데이터 간의 형변환 -> casting

** 컴퓨터는 왜 이진수를 쓸까 ? 
- 컴퓨터는 실리콘이라는 재료로 만든 반도체로 구성
-> 도체와 부도체에 반해 반도체는 전류로 제어가 가능.
   전류가 흐를 때 1, 흐르지 않을 때 0으로만 숫자를 표현할 수 있음
- 이진수 한자리 수 -> 1 bit 
8 bit -> 1 byte
1,024 byte -> 1 KB
1,025 KB -> 1 MB
1,024 MB -> 1 GB,,,

- ctrl + shift + N : 크롬 시크릿 모드
